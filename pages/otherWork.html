Nebula

Role: Solo Tech Artist

Software Used: Blender

Timeframe: Aug. 2022 - Ongoing

This is my procedural nebula project, having seen so far 5 primary iterations over the roughly 2 years I've been working on it. The general process is: 1. Define where the gas is coming from, and what colours should be emitted when/where. This is usually done via blender's geonodes in one way or another, but ultimately varies based on what ideas I want to try on a given iteration. 2. Define a domain, and any gravity emission objects we want, such as a black hole, or a neutron star that may have caused the nebula. 3. Simulate the gasses. 4. Use the simulation to define where stars should go. 5. Add shaders and render.

A background shader offshoot of this project (produced about a month before I began iteration 5) can be seen as the background of Overkill by Merlyn!

Mountain Background

Role: Solo Tech Artist

Software Used: Blender

Timeframe: Nov 2025 - Ongoing

I needed a background piece for multiple projects, and a snowy mountain seemed like the perfect fit.

I ended up using 5 A.N.T. terrain pieces to create the mountains, some quick dyntopo sculpting to tweak a few things, and a procedural shader material used to add snow.

This is still in progress. The snow needs work, and the surrounding terrain still feels too patterned.

In order to implement them I will likely have to split some of them up with DIY LoDs and bake some of the textures.

TM2020 Mapping: crimson Silence

Role: Solo Mapper

Software Used: TM2020 Editor

Timeframe: July. 2024 - Jan 2025

"crimson Silence" is a map I made for a weekly TM2020 speedmapping event, while also attempting to build up a series of maps trying to focus on clean but precise flow, while fitting the requirements of the event. This particular event required 1: Stadium car only 2: A slow section of the wood house blocks (1-4-1-x), 3: Use of "sausage blocks" (1-3-x).

This project is released, but the series is not finished. The map is available on TMX: (click here)
